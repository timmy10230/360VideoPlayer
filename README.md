# 360VideoPlayer
 
 此專案使用Unity開發，分成了電腦端以及手機端，使用者可使用電腦端登入google帳號查看雲端硬碟中的檔案，並將選擇的影片或圖片檔編排後傳入此系統的帳戶中，在手機端上進行查看，使用者也可以直接使用youtube網址，上傳想要的影片至系統中。
